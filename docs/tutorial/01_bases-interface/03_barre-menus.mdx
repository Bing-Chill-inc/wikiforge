La barre de menus est située en haut de la fenêtre de l'application. Elle contient les menus suivants :

-   **Fichier**
    -   **Nouveau**: Permet de créer un nouvel algorithme.
    -   **Ouvrir**: Permet d'ouvrir un algorithme existant.
    -   **Créer une copie**: Permet de créer une copie de l'algorithme actuel.
    -   **Partager**: Permet de partager l'algorithme avec d'autres utilisateurs.
    -   **Renommer**: Permet de renommer l'algorithme actuel.
    -   **Exporter**: Permet d'exporter l'algorithme sous différents formats.
    -   **Supprimer**: Permet de supprimer l'algorithme actuel.
-   **Édition**
    -   **Importer**: Permet d'importer un algorithme depuis un fichier.
    -   **Annuler**: Permet d'annuler la dernière action.
    -   **Rétablir**: Permet de rétablir la dernière action annulée.
    -   **Couper**: Permet de couper un élément sélectionné.
    -   **Copier**: Permet de copier un élément sélectionné.
    -   **Coller**: Permet de coller un élément copié ou coupé.
    -   **Sélectionner tout**: Permet de sélectionner tous les éléments de l'algorithme.
    -   **Supprimer**: Permet de supprimer un élément sélectionné.
    -   **Rechercher**: Permet de rechercher un élément dans l'algorithme.
-   **Aide**
    -   **Tutoriels**: Permet d'accéder aux tutoriels d'utilisation de l'application. (ici)
    -   **Raccourcis clavier**: Affiche les raccourcis clavier disponibles dans l'application.
    -   **À propos**: Affiche des informations sur l'application et ses développeurs.
