# Modifier des éléments

## Problèmes

<Probleme titre="Inverser 2 nombres" données="nbr1, nbr2" />

Pour modifier le titre d'un problème, il suffit de cliquer sur le problème en question. 
Une fois le curseur placé dans le problème, vous pouvez modifier le titre.

Pour modifier ses données / résultats, cliquez sur les emplacements de données / résultats.
S'il n'y a pas encore de données / résultats, survoler la zone de données / résultats avec la souris les fera apparaître.

## Structure Alternative Si

<StructureSi conditions={["note < 10", "Sinon"]} max={5} />

Une structure alternative est composée de plusieurs conditions. Pour modifier une condition, cliquez sur la condition en question.
Une fois le curseur placé dans la condition, vous pouvez modifier le texte.

Pour ajouter une condition, survolez le côté droit de la structure alternative. Pour y faire apparaître le bouton <code>+</code>. Cliquez dessus.
Pour ajouter une condition à un emplacement différent, cliquez sur le bouton <code>+</code> de la condition précédente (apparait au survol).
Pour supprimer une condition, cliquez sur le bouton <code>-</code> de la condition (apparait au survol).
Note: Vous ne pouvez pas supprimer la dernière condition.

Pour échanger deux conditions, cliquez sur les boutons de déplacement en bas de la condition (apparaissent au survol).

## Structure Alternative Switch

<StructureSwitch conditions={["bonjour", "camion", "algoforge", "hélicoptère"]} max={5} variable="mot" />

Une structure alternative switch est composée d'une expression à tester (en haut) et de plusieurs cas possibles (en bas).
Pour modifier l'expression à tester, cliquez sur l'expression en question.
Une fois le curseur placé dans l'expression, vous pouvez modifier le texte.

Pour modifier un cas, cliquez sur le cas en question.
Une fois le curseur placé dans le cas, vous pouvez modifier le texte.

Pour ajouter un cas, survolez le côté droit de la structure alternative switch. Pour y faire apparaître le bouton <code>+</code>. Cliquez dessus.
Pour ajouter un cas à un emplacement différent, cliquez sur le bouton <code>+</code> du cas précédent (apparait au survol).
Pour supprimer un cas, cliquez sur le bouton <code>-</code> du cas (apparait au survol).
Note: Vous ne pouvez pas supprimer le dernier cas.

Pour échanger deux cas, cliquez sur les boutons de déplacement en bas du cas (apparaissent au survol).

## Structures Itératives

<StructureIterative variable="i" pas="1" borneInf="1" borneSup="10" ordre="croissant" estBornée />

En double cliquant sur une structure itérative, on peut la modifier. On peut alors paramétrer l'**indice à itérer**, 
le **pas** et sa **direction** (croissant ou décroissant), ainsi les **borne inférieure** et **supérieure**.
On peut aussi choisir l'option "Non bornée", pour que la boucle ne soit d'un nombre d'itérations fixe.
(Note: Pour les boucles non bornées, pensez à ajouter une condition d'arrêt pour éviter les boucles infinies.)../../src/components/AlgoforgeElements/ElementsGraphiques/StructureIterativeNonBorne/structure-iterative-non-bornee
