# Déplacer des éléments

Pour déplacer un élément, cliquez dessus et maintenez le clic enfoncé. Déplacez ensuite l'élément à l'endroit souhaité.

Pour faciliter le déplacement, vous pouvez aussi sélectionner l'élément en cliquant dessus, puis utiliser les flèches de votre clavier pour le déplacer.

Vous pouvez déplacer plusieurs éléments en même temps en les sélectionnant. Pour cela, cliquez sur un espace vide du plan de travail et maintenez le clic enfoncé. 
Faites glisser votre souris pour sélectionner les éléments à déplacer. Relâchez le clic pour valider la sélection. 
Vous pouvez ensuite déplacer les éléments sélectionnés en les faisant glisser.

Vous pouvez aussi sélectionner plusieurs éléments en utilisant la touche Maj / Shift de votre clavier.

## Aligner des éléments

Pour aligner des éléments entre eux, sélectionnez les éléments à aligner (avec sélection rectangle ou shift + clic). 
Ensuite, effectuez un clic droit sur l'élément que vous souhaitez utiliser comme référence pour l'alignement.
Dans le menu contextuel, cliquez sur "Aligner la sélection ici" pour aligner les éléments sélectionnés sur l'élément de référence.
