# Introduction

L'Éditeur AlgoForge est basé sur le paradigme des composants.
Il est conçu pour être facile à utiliser et à comprendre, tout en offrant une grande flexibilité pour les utilisateurs avancés.

## Structure de dossiers de l'éditeur

<GitHubFolderStructure githubUrl="https://github.com/Bing-Chill-inc/Algoforge-main" pathToDisplay="src/front-editeur" />

Les fichiers composants la partie frontend de l'éditeur sont organisés dans le dossier `src`.

Le fichier `.gitignore` sont utilisés pour ignorer certains fichiers lors de la gestion du code source.

Le fichier [`SmeltJS.ts`](./07_smeltjs.mdx) est le bundler custom utilisé par AlgoForge.
