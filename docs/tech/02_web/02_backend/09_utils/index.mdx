# Utils

## Description Générale

Le dossier `utils/` contient des fonctions et classes utilitaires utilisées dans l'application backend d'AlgoForge. Ces utilitaires sont conçus pour centraliser des fonctionnalités transversales, comme la validation des données, la gestion des logs, le hashage des mots de passe, ou encore la génération de tokens. Ils permettent de réduire la duplication de code et d'améliorer la maintenabilité du projet.
